# Critérios de Aprovação de Artigos para a Base de Conhecimento

Este documento detalha os critérios para a curadoria e aprovação de artigos científicos a serem incluídos na base de conhecimento. O objetivo é garantir que a base seja composta por documentos de alta qualidade, relevância e riqueza de informações, para que possa ser utilizada como uma fonte confiável para treinar outros modelos de linguagem (LLMs) e para consulta.

## 1. Qualidade do Conteúdo

A qualidade do conteúdo é o critério mais importante. Artigos de baixa qualidade, com informações imprecisas ou mal-escritos, serão rejeitados.

*   **Rigor Científico:** O artigo deve demonstrar rigor científico, com metodologia clara, resultados bem-definidos e conclusões baseadas em evidências.
*   **Clareza e Coerência:** O texto deve ser claro, coerente e bem-escrito, sem ambiguidades ou contradições internas.
*   **Fonte Confiável:** O artigo deve ser de uma fonte confiável, como um periódico científico revisado por pares, uma conferência de renome ou uma instituição de pesquisa reconhecida.
*   **Originalidade:** O artigo deve apresentar uma contribuição original para a área de conhecimento, seja através de novos resultados, uma nova metodologia ou uma nova perspectiva sobre um problema existente.

## 2. Relevância para a Base de Conhecimento

A base de conhecimento é focada em agronomia. Portanto, os artigos devem ser relevantes para esta área.

*   **Tópico Principal:** O foco principal do artigo deve ser agronomia prática, como cultivo de culturas, manejo do solo, controle de pragas e doenças, irrigação, fertilização, etc.
*   **Relevância Prática:** O artigo deve ter relevância prática para a agronomia, com resultados que possam ser aplicados no campo ou que contribuam para o avanço da tecnologia agrícola.
*   **Exclusão de Tópicos Genéricos:** Artigos sobre biologia geral, química, ciência climática ou outras áreas, onde a agronomia é apenas um exemplo menor, serão rejeitados.

## 3. Estrutura e Formato

A estrutura e o formato do artigo são importantes para garantir a extração correta das informações.

*   **Formato de Artigo Científico:** O documento deve ser um artigo científico completo, tese ou estudo de caso detalhado, com seções claras de Introdução, Metodologia, Resultados e Conclusão/Discussão.
*   **Exclusão de Formatos Inadequados:** Resumos, notícias, artigos de opinião, conteúdo de marketing ou qualquer outro formato que não seja um artigo científico completo será rejeitado.
*   **Legibilidade:** O texto deve ser legível, sem problemas de formatação que dificultem a leitura ou a extração de informações.

## 4. Qualidade dos Dados

A qualidade dos dados apresentados no artigo é crucial para a confiabilidade da base de conhecimento.

*   **Dados Claros e Acessíveis:** Os dados devem ser apresentados de forma clara e acessível, preferencialmente em tabelas ou gráficos bem-definidos.
*   **Resultados Mensuráveis:** Os resultados devem ser mensuráveis e quantificáveis, com unidades de medida claras e análise estatística apropriada, quando aplicável.
*   **Disponibilidade de Dados Suplementares:** A disponibilidade de dados suplementares, como conjuntos de dados brutos, é um fator positivo, mas não obrigatório.

## Processo de Curadoria

O processo de curadoria é realizado por um modelo de linguagem treinado para seguir estes critérios. O modelo preenche uma série de campos de metadados extraídos do artigo e, com base nos critérios acima, decide se o artigo deve ser aprovado ou rejeitado, fornecendo um feedback claro e justificado para a sua decisão.
